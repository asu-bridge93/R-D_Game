ゲームの説明
基本ルール

あなたは企業の役割を担い、R&D投資の意思決定を行います。 このゲームでは、12人の参加者が4人ずつのグループに分かれます。

毎年の流れ
  • 毎年、あなたには5枚のカードが配られます。1枚のカードは{{ card_value }}億円の価値があります。
  • あなたはこれらのカードから何枚をR&D投資に使うか決定します（0〜{{ cards_per_player }}枚）。
  • グループ内の4人が投資したカードの合計枚数によって、R&Dの成功確率が決まります。
R&Dの成功確率
  • グループの合計カード枚数が0〜4枚：成功確率0%
  • グループの合計カード枚数が5〜11枚：成功確率33.3%
  • グループの合計カード枚数が12〜16枚：成功確率50%
  • グループの合計カード枚数が17〜20枚：成功確率66.7%
R&Dが成功した場合
  • グループ内でR&Dに成功した企業が決まります。
  • あなたが投資したカードの枚数が多いほど、成功企業に選ばれる確率が高くなります。
  • 現在の条件： {% if is_winner_takes_all %} Winner Takes All {% else %} Spill Over {% endif %}

ゲームの進行

R&Dに成功した場合、利益が確定し次の商品開発に移ります。
R&Dに失敗した場合、次年度の投資決定に移ります。
最大5回の投資決定を行います。
